# docker-postgres

Running the postgres:15.3-alpine3.18 with dpage/pgadmin4:7.1 image:

````bash
docker-compose up
````

Access [http://127.0.0.1:5050](http://127.0.0.1:5050)

• Pgadmin4:
  • host: http://127.0.0.1:5050
  • user: pgadmin4
  • passwd: pgadmin4_passwd

• Postgres:
  • host: postgres
  • user: pgadmin4
  • passwd: pgadmin4_passwd
